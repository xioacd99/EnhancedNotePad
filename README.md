## 如何参与项目？

### 提建议

Issues 部分新建一个 Issue，填写对应的标题、描述、Labels、（Assignees）

### 完善文档

将项目 clone 到本地进行修改，pull 时选择 pull merge 而不是直接 commit to master。您的贡献将经过审核判断是否接受。

所有文档使用 markdown 编写，放在 documents 文件夹下，文档中引用的图片放在 image 文件夹下（相对路径）。
